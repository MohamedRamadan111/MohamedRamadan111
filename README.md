## Architecture

**RAG Pipeline**

```
User Query  →  Embedding Model  →  Vector DB
                                       ↓
              Final Response  ←  LLM  ←  Re-ranker  ←  Hybrid Retriever
                                                         (Semantic + BM25 + RRF)
```

**AI Agent Workflow**

```
User Task  →  Planner Agent  →  Tool Selection  →  Execution Agent  →  Memory  →  Response
```
